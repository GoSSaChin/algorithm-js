## 문제 해석

- 주어진 수 `n`보다 작은 자연수로 `n`을 나눴을 때 나머지가 1인 가장 작은 자연수 구하기

## 최종 제출 코드

```js
function solution(n) {
  var answer = 0;

  for (let i = 1; i < n; i++) {
    if (n % i === 1) {
      answer = i;
      break;
    } else {
      continue;
    }
  }
  return answer;
}

solution(10);
```

## 시간 복잡도

- **∴ O(n)**
- for문을 도는 루프가 `n-1`번 반복하기 때문에 `O(n)`
