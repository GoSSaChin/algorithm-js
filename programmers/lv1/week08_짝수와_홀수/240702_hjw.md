## 문제 해석

- 주어진 정수 `num`이 짝수일 경우 "Even"을, 홀수일 경우 "Odd" 반환

<br/>

## 최종 제출 코드

```javascript
function solution(num) {
  return num % 2 === 0 ? "Even" : "Odd"
}
```

- `num`을 `2`로 나눴을 때 나머지가 0이면 `even`, 1이면 `odd`

<br/>

## 시간 복잡도

- **∴ O(1)**
