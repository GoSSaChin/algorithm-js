## 문제 해석

- 주어진 문자열을 정수로 변환

<br/>

## 최종 제출 코드

```javascript
function solution(s) {
  return Number(s);
}
```

- `Number`를 이용하여 문자열 `s`를 정수로 변환하여 return

<br/>

## 시간 복잡도

- 시간 복잡도는 변환할 문자열의 길이에 비례함
- `n`: `s`의 길이
- **∴ O(n)**
