## 문제 해석

-   주어지는 문자열을 숫자로 반환하는 문제

## 최종 제출 코드

-   문자열 s를 숫자로 변환하여 반환

```js
function solution(s) {
    return Number(s);
}
```

## 시간 복잡도

-   Number: 문자열을 분석하여 유효한 숫자로 변환 O(n). 여기서 n은 문자열 s의 길이
-   **∴ O(n)**
